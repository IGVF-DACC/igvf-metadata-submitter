Drag and drop a root folder that contains all of your files to be submitted. You cannot drop multiple folders or files.

For each file, define #upload_relpath (including the root folder) on the sheet.

For example, you drop a root folder test-folder then a file test-folder/1/2/3.fastq.gz should be defined under #upload_relpath on the sheet.

Only files matching relative paths will be available for uploading.
